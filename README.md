Questa repository include le esercitazioni svolte per il laboratorio di simulazione numerica. Ogni cartella (`LabXX`) corrisponde ad una esercitazione e al suo interno sono presenti:
- delle sottocartelle (`esercizioXX.Y`) contenenti i codici C++ e i file di output dei singoli esercizi
- il Jupyter Notebook relativo all'esercitazione (`nb_LabXX.ipynb`)


Di seguito, viene riportato un elenco dei principali argomenti trattati nelle varie esercitazioni: 
- **Esercitazione 1**: metodi Monte Carlo e tecnica del data blocking per l’analisi statistica degli errori.
- **Esercitazione 2**: importance sampling e random walk.
- **Esercitazione 3**: integrazione stocastica e valutazione di opzioni finanziarie.
- **Esercitazione 4**: dinamica molecolare.
- **Esercitazione 6**: modello di Ising.
- **Esercitazione 7**: dinamica molecolare e tail corrections.
- **Esercitazione 8**: principio variazionale e algoritmo di simulated annealing.
- **Esercitazione 9**: Travelling Salesman Problem e algoritmi genetici.
- **Esercitazione 10**: parallelizzazione del codice e tecniche di calcolo parallelo.
- **Esercitazione 11**: Deep Neural Networks.
- **Esercitazione 12**: Convolutional Neural Networks.
